# mithril-kata Mithril Forms

Validation of form fields using PowerForm, https://github.com/ludbek/powerform


This project was created using **webpack-cli** command:

```
npx webpack-cli init
```

*Versions:*
- Mithril: 1.1.6
- node: v18.10.0
- webpack: @5.0.0

You can run the example:

```
npm run watch
```
